## 3. Palindrome Check (Recursive)

### Problem Name
Palindrome Check

### Problem Description
Given a string \(S\), determine whether it is a **palindrome** using recursion.

A palindrome reads the same forward and backward.

### Input Description
- A single string \(S\)

### Output Description
- Print `YES` if the string is a palindrome, otherwise print `NO`

### Sample Test Cases
**Input**
```
level
```
**Output**
```
YES
```

**Input**
```
codeforces
```
**Output**
```
NO
```

### Constraints
- \(1 \le |S| \le 10^5\)
- String contains lowercase English letters

---
##JAVA SOLUTION
import java.util.Scanner;
class palindrome{
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        System.out.println("Enter no of queries");
        int q=sc.nextInt();
        while(q-->0){
        System.out.println("Enter any String");
        String s=sc.next();
        System.out.println(palin(s,0,s.length()-1));
        }
        sc.close();
    }
    public static boolean palin(String s,int left,int right) {
        if(left==right){
            return true;
        }
        if(s.charAt(right)!=s.charAt(left)){
            return false;
        }
        return palin(s, left+1, right-1);
    }
}
